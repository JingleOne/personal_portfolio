The essence of this project

How backtracking search algorithm works

The backtracking algorithm works by trying out the possible candidate at each step, in this project, number 1-9 at every empty grid that needs to be filled in. When it tries out a number, if it violated the rule of sudoku, then it will attempt the next number from 1-9. If none of the number works, that means one or some of the previous answers were wrong, then it will roll back to the last grid and try out the next number.
