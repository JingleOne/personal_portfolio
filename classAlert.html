Class Notifier

A really simple program that uses selenium and Twilio API to help me get class info instantly on my phone

Intention of this program

Every time when the classes enrollment opened up, the classes I wanted were always full before I got to choose my desired classes. Therefore, I have to wait in front of the computer tracking the enrollment status and the waitlist. To alleviate this pain, I created this very simple program. I just need to put in the classes information, and the program will check the status for me once every few minutes. If a class spot is available, then a message will send to my phone through Twilio API.
